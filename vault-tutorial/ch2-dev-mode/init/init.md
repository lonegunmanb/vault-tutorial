# 实验：Dev 模式的安全边界与风险亲手复现

在本实验中，你将**主动地**把 Dev 模式的三个核心安全缺陷变成可见的现实：

1. **数据短暂性**：写入一个机密，重启 Dev 服务器，亲眼目睹机密凭空消失
2. **明文流量**：用 `tcpdump` 嗅探本地回环网络，在网络包中直接读出你的 Vault Token 和机密值
3. **Root Token 的公开暴露**：用 `ps` 命令从进程列表中"偷取"另一个用户的 Vault Root Token

这不是模拟——这是在你本地机器上真实发生的安全事件。

理解了这些缺陷，你会在职业生涯中永远不会"顺手"把 Dev 模式用于生产。
